электронная курилка
для самых крутых

Разноцветные одноразовые ашкудишки наскучили и приелись? Даже вкус курицы гриль больше не удивляет? Представляем SWAP — бесконечную электронную курилку с кастомизацией ароматов.

Ключевая технология в основе SWAP — устройство синтеза ароматов, позволяющее реализовать любой запах, который только доступен человеческому носу.

Сегодня можно покурить лесные ягоды, завтра — яблочный спринг-ролл из чихо, послезавтра — бутерброд с колбасой. возможности безграничны: тебя остановит только твоя фантазия.
